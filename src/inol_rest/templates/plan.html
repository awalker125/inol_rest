Examples

In order to get strong you must lift heavy weights but it is the recovery that actually makes you stronger. If you do not lift enough there will be insufficent stimulus to cause the body to adapt. If you lift to much you will not be able to recover and actually end up weaker or more likely injured. So how should you design a training program that works without wasting time or getting injured?

Russian Squat Program

The Russian squat program found here is a three days per week program over six weeks which promises to increase your max by 5%. The hardest sessions are normally 6x6@80% and 3x3@95%



Day 1 Day 2 Day 3 Weekly Inol
Week 1 6x2@80% N = 0.6 6x3@80% N = 0.9 6x2@80% N = 0.6 w = 2.1
Week 2 6x4@80% N = 1.2 6x2@80% N = 0.6 6x5@80% N = 1.5 w = 3.3
Week 3 6x2@80% N = 0.6 6x6@80% N = 1.8 6x2@80% N = 0.6 w = 3
Week 4 5x5@85% N = 1.6 6x2@80% N = 0.6 4x4@90% N = 1.6 w = 3.8
Week 5 6x2@80% N = 0.6 3x3@95% N = 1.8 6x2@80% N = 0.6 w = 3
Week 6 2x2@100% N = 4** 6x2@80% N = 0.6 1x1@105% N = 2** w = 6.6**

** inols can not be calculated for intensities of 100% or above. Here we assume this is actually 99%
